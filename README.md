# To‑Do Lista

Vježbena web aplikacija za upravljanje zadacima.  
Omogućuje dodavanje, pregled, uređivanje i brisanje zadataka.

---

## Tehnologije

- HTML / CSS / JavaScript  
- PHP  
- MySQL (ako je korištena baza)

---

## Funkcionalnosti

- Dodavanje novih zadataka  
- Pregled liste zadataka  
- Uređivanje i brisanje zadataka  

---

## Pokretanje

1. Kloniraj repozitorij: git clone https://github.com/AntonelaGalicc/To-Do-Lista.git
2. Pokreni lokalni server (XAMPP/WAMP/LAMP) i otvori `index.html` u pregledniku.  
3. Ako koristiš bazu, kreiraj tablicu za zadatke i ažuriraj konekciju u PHP skriptama.
